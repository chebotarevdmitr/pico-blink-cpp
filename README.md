Raspberry Pi Pico Blink (C++)

Добро пожаловать в мой первый проект на Raspberry Pi Pico! 🎉 Здесь мы учимся мигать светодиодом, но делаем это с серьёзным лицом и лёгким юмором.

📖 Описание проекта

Этот проект написан на C++ с использованием Raspberry Pi Pico SDK. Основная цель — заставить встроенный светодиод на плате мигать каждые полсекунды. Казалось бы, просто, но именно так начинается путь к великим делам в мире микроконтроллеров.

⚙️ Сборка проекта

Установите Pico SDK и зависимости (CMake, GCC ARM Toolchain).

Клонируйте этот репозиторий:

git clone https://github.com/chebotarevdmitr/pico-blink-cpp.git
cd pico-blink-cpp

Соберите проект:

mkdir build
cd build
cmake ..
make

После сборки появится файл blink.uf2.

🔌 Прошивка на Pico

Отключите Pico от USB.

Зажмите кнопку BOOTSEL и подключите Pico к USB.

На компьютере появится диск RPI-RP2.

Скопируйте туда файл blink.uf2.

Отпустите кнопку и наблюдайте, как светодиод начинает мигать. ✨

🖥️ Если у вас нет Pico

Не беда! Вы всё равно можете:

Собрать проект и убедиться, что SDK и CMake работают.

Изучить код и структуру проекта.

Использовать эмуляторы RP2040 (например, QEMU-RP2040).

Сохранить проект на GitHub и прошить его позже, когда плата появится.
